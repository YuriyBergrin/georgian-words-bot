# Georgian Words Bot

Telegram-бот для изучения грузинских слов на `aiogram 3` с PostgreSQL, Redis и Alembic.

## Что уже реализовано

- регистрация пользователя по `/start` (без дублей по `telegram_id`)
- главное меню (`ReplyKeyboardMarkup`)
- добавление слова через FSM
- простая тренировка слов:
  - случайное слово из БД
  - проверка перевода без учета регистра и лишних пробелов
  - переход к следующему слову или возврат в меню

## Стек

- Python 3.13+
- aiogram 3
- SQLAlchemy (async)
- asyncpg
- Redis
- Alembic
- Loguru

## Быстрый старт

1. Создать и активировать виртуальное окружение.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Заполнить `.env` (можно взять `.env.example`):

- `BOT_TOKEN`
- `DATABASE_URL`
- `REDIS_URL`

4. Поднять инфраструктуру (если используешь docker):

```bash
docker compose up -d
```

5. Применить миграции:

```bash
alembic upgrade head
```

6. Запустить бота:

```bash
python main.py
```

## Workflow миграций

Используем согласованный процесс:

1. Изменить модели
2. Сгенерировать миграцию:

```bash
alembic revision --autogenerate -m "..."
```

3. Проверить файл миграции вручную
4. Применить:

```bash
alembic upgrade head
```
