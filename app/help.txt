📘 Инструкция по боту Georgian Words Bot

Основные команды:
- /start — запуск бота и открытие главного меню
- /help — показать эту инструкцию

Как учить слова:
1. Нажми «📚 Учить слова».
2. Выбери режим:
   - «🌍 Все слова» — тренировка по всем словам
   - «📂 Выбрать тему» — тренировка по выбранной теме
3. Отвечай на вопросы:
   - часть вопросов с вариантами ответа
   - часть — вводом текста
4. Для выхода из тренировки нажми «🔙 Назад в меню».

Как работает повторение:
- Бот ведет прогресс по каждому слову и направлению перевода.
- Слова, которые нужно повторить, появляются автоматически по расписанию.
- Если ошибся, слово повторится снова через короткий интервал.

Админ-функции (только для администратора):
- «➕ Добавить слово» — добавить слово вручную
- «📥 Импорт слов» — массовый импорт строк:
  georgian | russian | topic
- «✏️ Редактировать слово» — изменить перевод и тему
- «🔥 Топ сложных слов» — слова с наибольшим числом ошибок
